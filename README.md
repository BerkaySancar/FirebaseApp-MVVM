# FirebaseApp-MVVM

## Features

- Swift UIKit
- Storyboard
- Auto Layout
- CocoaPods
- Firebase Authentication & Firestore Databese & Storage
- Sign in with Apple
- UITableView
- PHPickerViewController

## Architecture 

Protocol-Oriented MVVM.

### Before the build and run

File -> Packages -> Reset Package Caches. After this process you can build and run.
